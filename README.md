# Hand Gesture Controlled Mouse 🖐️🖱️

A computer vision–based virtual mouse that allows users to control the system cursor using hand gestures captured through a webcam.  
The project uses MediaPipe Hand Tracking, OpenCV, and PyAutoGUI to translate finger movements into real-time mouse actions.

The system detects hand landmarks from the webcam feed and maps them to screen coordinates, enabling interactions such as cursor movement and mouse clicks.
